# VoiSe Gate 5.3 — SoundBoard Layout Fixes

Gate 5.3 дорабатывает SoundBoard после визуальной проверки Gate 5.2.

## Что изменено после Gate 5.2

- заголовок окна: `VoiSe Gate 5.3 - SoundBoard Layout Fixes`;
- кнопки транспорта теперь образуют компактный блок:
  - верхняя строка: предыдущий / следующий трек;
  - нижняя строка: `Stop` на ширину двух верхних кнопок;
- вместо emoji используются системные glyph-символы Segoe MDL2 Assets;
- `Play/Pause` остаётся отдельной кнопкой слева от таймлайна и меняет glyph:
  - Play: `E768`;
  - Pause: `E769`;
- таймлайн теперь выполняет seek во время изменения значения, а не только по PointerReleased;
- выпадающий список категорий занимает примерно 25% ширины окна и остаётся слева;
- увеличен визуальный промежуток между выбором категории и списком треков;
- нижняя правая панель теперь зависит от вкладки:
  - `SoundBoard`: статистика категорий/звуков/использования;
  - `Settings`: лог приложения;
  - `Voice Changer` и `Scenes`: пустое место.

## Запуск

```powershell
dotnet run --project src/VoiSe.App
```

## Проверка

1. В заголовке окна должно быть `VoiSe Gate 5.3`.
2. На SoundBoard кнопки предыдущий/следующий/Stop должны образовывать прямоугольный блок.
3. Play/Pause должна быть отдельной кнопкой слева от таймлайна.
4. Запусти трек и перемести таймлайн — звук должен перейти к выбранному месту и не возвращаться назад.
5. Выпадающий список категории должен быть узким и располагаться слева.
6. На вкладке SoundBoard внизу справа должна быть статистика.
7. На вкладках Voice Changer и Scenes внизу справа должно быть пусто.
8. На вкладке Settings внизу справа должен отображаться лог.
