# VoiSee 10.5.0 — Interim Release

## Purpose

This archive is the project checkpoint immediately before development of the
Yandex Music integration. It keeps the accepted VoiSee 10.4.1 behavior and
promotes the application metadata to version 10.5.0.

## Change in this release

- Removed the `Edit Track` button from the SoundBoard list toolbar.
- Kept `Edit Sound` in each track's context menu.
- Kept `Add Track` and `Delete Track` in the list toolbar.
- No Yandex Music integration code, credentials, services, or dependencies are
  included yet.

## Validation status

No smoke tests, automated checks, or Windows build were run for this checkpoint,
per the project owner's instruction. The archive is intended for the owner's
normal Windows build and manual acceptance.
