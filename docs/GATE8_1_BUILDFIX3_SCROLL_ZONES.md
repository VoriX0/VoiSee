# Gate 8.1 buildfix 3 — Scroll zone tuning

This buildfix contains only wheel routing and log dialog adjustments.

## Changes

- Voice preset icon picker wheel zone is shifted to the right by 15%.
- Settings wheel zone is extended to the right by 50%.
- Scene sound buttons wheel zone is extended to the right by 50%.
- Scene list wheel zone is extended to the right by 15%.
- Log dialog height is reduced to avoid clipping the bottom of the log content.
- Log dialog now opens scrolled to the latest log entries.

## Not changed

- Audio runtime logic.
- Scene playback logic.
- SoundBoard playback logic.
- Voice preset icon list and visual design.
