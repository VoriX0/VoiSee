# Gate 6.17 — Hotkey Capture UX

- Replaces manual hotkey typing with press-to-capture fields.
- Combines SoundBoard Play/Pause into one transport hotkey.
- Keeps bare single-key hotkeys local-only outside VoiSe, so typing in other apps is not blocked.
- Modifier hotkeys remain global and are consumed by VoiSe when matched.
