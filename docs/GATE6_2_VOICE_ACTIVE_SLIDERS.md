# Gate 6.2 — Voice Active Sliders

This gate narrows Voice Changer to parameters that are actually wired into the current audio processor.

## Active sliders

- Voice Gain
- Gate
- Compressor

## Removed from current UI

The following sliders are removed until a stronger DSP layer is added:

- Input Gain
- Pitch
- Formant
- Compression Ratio
- Limiter
- Robot
- Radio
- Reverb
- Brightness

The app keeps older settings fields for compatibility, but the UI and preset writer no longer use them.

## SoundBoard tuning

The SoundBoard wheel zone is expanded further to the right and down while keeping the existing layout unchanged.
