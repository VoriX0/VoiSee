# VoiSee 11.0.5

Media Bridge meter refinement:

- Source and To Mic meters are positioned closer together.
- Meter UI refresh interval is reduced from 200 ms to 50 ms.
- Preview refresh remains approximately once per second.
- Meter movement now uses a -60..0 dBFS scale.
- Added dBFS tick labels and guide marks.
- Current values below the meters are shown in dBFS.
- No Media Bridge routing or capture behavior was changed.

Build and automated tests were not run at the user's request.
