# VoiSee 11.0.1 — Media Bridge UI refinement

- Preview moved to the left and the window is fitted without cropping.
- Click the preview itself to select or replace the source window.
- Added centered plus affordances for empty and selected preview states.
- Controls and source settings moved to a dedicated right-side panel.
- Start/Pause/Resume are now one `Play / Pause` button.
- Stop and Play/Pause sit at the top of the right panel and are hidden until a source is selected.
- Source level is displayed vertically.
- Future Ducking is simplified to one strength slider; Attack, Release and fades are removed from the product specification.

No build or tests were run for this archive at the user's request.
