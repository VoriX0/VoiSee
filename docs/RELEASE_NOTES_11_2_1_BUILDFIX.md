# VoiSee 11.2.1 Buildfix

Исправлена ошибка сборки VoiSee 11.2.0.

## Исправления

- В `WarmSoundCacheInBackground(...)`, который возвращает `void`, ошибочный `return false;` заменён на `return;`.
- Сравнения `MainTabView.SelectedItem` с `MediaBridgeTabViewItem` заменены на явный `ReferenceEquals(...)`, чтобы убрать предупреждения CS0252 и подчеркнуть проверку конкретного экземпляра вкладки.
- Версия проекта обновлена до 11.2.1.

По представленному логу `MSB3073` от XamlCompiler выглядит каскадной ошибкой после CS0127. Отдельных сообщений об ошибке XAML в логе нет, поэтому XAML-разметка не изменялась.

Сборка и тесты не запускались по указанию пользователя.
